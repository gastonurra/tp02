¿que es una URL?

una URL (Uniform Resource Locator) es la dirección que se utiliza para acceder a un recurso en Internet. Indica la ubicación de un recurso específico y el protocolo que se debe utilizar para acceder a el, como HTTP, HTTPS, FTP.

¿que es una URL absoluta?

una URL absoluta es una dirección completa que especifica el protocolo y la ruta completa desde el servidor raíz hasta el recurso. Incluye el nombre de dominio y cualquier subdirectorio necesario para llegar al recurso.

¿que es una URL relativa?

una URL relativa es una dirección que se especifica en relación con la posición actual del documento en el que se está utilizando. No incluye el nombre de dominio y es útil para enlazar recursos dentro del mismo sitio web.

¿cual es la diferencia entre una URL relativa y una URL absoluta?

la diferencia principal entre una URL relativa y una URL absoluta es que la URL absoluta incluye el protocolo y el nombre de dominio completo, mientras que la URL relativa no lo hace. Una URL relativa es útil para enlazar recursos dentro del mismo dominio sin necesidad de especificar el nombre completo del dominio.

¿que es un directorio? ¿que relación hay con una carpeta?

un directorio es una estructura de organización utilizada para almacenar y gestionar archivos en un sistema de archivos. Un directorio es similar a una carpeta en un sistema operativo gráfico y se utiliza para agrupar archivos relacionados.

en la estructura de un proyecto web, ¿a qué se llama directorio raiz? ¿con qué simbolo se representa?

en un proyecto web, el directorio raíz es la carpeta principal que contiene todos los archivos y subdirectorios del proyecto. Se representa con el símbolo "/" (barra inclinada) y es el punto de partida para las rutas absolutas en un servidor web.

¿cual es el propósito de la etiqueta anchor?

el propósito de la etiqueta anchor (<a>) es crear enlaces a otros documentos o ubicaciones en la web. permite a los usuarios navegar entre páginas web o descargar recursos específicos.

¿que atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

el atributo "href" se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar.

¿que sucede si el atributo "href" en la etiqueta anchor está vacío?

si el atributo "href" en la etiqueta anchor está vacío, el enlace no redireccionará a ninguna página. Dependiendo del contexto, puede causar un comportamiento inesperado, como recargar la misma página.

¿cual es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

el atributo "target" en la etiqueta anchor se utiliza para especificar dónde se abrirá el enlace. Los posibles valores son:

  • _self: abre el enlace en la misma ventana o pestaña (valor por defecto).
  • _blank: abre el enlace en una nueva ventana o pestaña.
  • _parent: abre el enlace en el marco padre (si existe).
  • _top: abre el enlace en la ventana completa, reemplazando cualquier marco.
